# Calculator

## Aim : 
The aim of the project is to create a Calculator App using Java Swing.
It perform basic operations:
- Addition
- Multiplication
- Division
- Subtraction
- Square-
- Square root
 
## Languages Used :
- Java 
- Java Swing
